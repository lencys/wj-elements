WebJet Elements ponúka jednoduchú integráciu animácií do webových aplikácií pomocou elementu `Animation`, čím zjednodušuje vytváranie pútavých používateľských rozhraní. Poskytuje možnosti prispôsobenia v podobe oneskorenia, trvania alebo napríklad tiež počtu opakovaní animácií.

Využíva k tomu rozsiahlu knižnicu CSS animácií [Animate.css](https://animate.style/), z ktorej si zvolíte želanú animáciu, napríklad `HeartBeat` a jej názov definujete v atribúte `name` podľa ukážky nižšie.

import Basic from '@site/static/usage/v1/animation/basic/index.md';

<Basic />

Pre viac informácii o možnostiach prispôsobenia sa presuňte do elementu [animation](/api/animation).
