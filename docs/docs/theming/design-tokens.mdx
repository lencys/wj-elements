WebJET Elements používa globálne CSS tokeny ako základ celej vizuálnej vrstvy. Namiesto izolovaného prepisovania
štýlov na komponentoch je efektívnejšie začať práve tu: od typografie, spacingu, farieb, radiusov a timingov.

Nasledujúci prehľad funguje ako centrálna referencia pre predvolenú tému a je vhodným štartovacím bodom pri tvorbe
vlastného brandingu alebo dizajn systému nad WebJET Elements.
