Tree Item (wje-tree-item) element predstavuje jednotlivú položku v rámci Tree komponentu. Môže obsahovať text, ikony alebo ďalšie vnorené položky, čím umožňuje vytvárať hierarchické štruktúry.

:::note Poznámka

Pre viac informácií o použití Tree item sa presuňte do dokumentácie elementu [**Tree**](./tree).

:::

## Kedy použiť

Použite `wje-tree-item`, keď používateľ potrebuje orientáciu v aplikácii alebo prechod medzi stavmi/obrazovkami.

## Kedy nepoužiť

Nepoužívajte viac paralelných navigačných vzorov, ktoré si navzájom konkurujú.

## Prístupnosť

Zabezpečte jasné active/selected stavy, predvídateľné poradie tabulátora a pomenovanie ovládacích prvkov.

## Odporúčané postupy

- Držte URL a UI stav v synchronizácii, aby bola navigácia reprodukovateľná.
- Používajte konzistentné názvoslovie položiek naprieč menu, breadcrumbom a tabmi.
- Pri hlbokých štruktúrach pridajte pomocný kontext (breadcrumb, nadpis, ikony).
- Pri 3+ úrovniach vnorenia dolaďte odsadenie cez `--wje-tree-item-indent`, aby boli úrovne vizuálne jednoznačné.
