Posuvníky sú užitočné interaktívne ovládacie prvky, ktoré umožňujú používateľom intuitívne a pohodlne vybrať hodnotu z daného rozsahu. Tieto vizuálne prvky poskytujú používateľom možnosť presného nastavenia parametrov, ako napríklad hlasitosť, jas obrazovky, časový údaj alebo iné hodnoty, ktoré je potrebné zmeniť.

Používateľ môže jednoducho ovládať posuvník pomocou interakcie s myšou, dotykovým displejom alebo inými vstupnými zariadeniami. Týmto spôsobom môže plynule posúvať posuvník po pásme alebo zónach a v reálnom čase sledovať zmenu hodnoty. Vizuálny prvok posuvníka poskytuje spätnú väzbu používateľovi, čím mu umožňuje presne určiť zvolenú hodnotu.

## Atributy elementu

### Value

Určuje počiatočnú hodnotu komponenty.

### Min
#### Minimálna hodnota

Najnižšia hodnota v rozsahu povolených hodnôt. Ak value je hodnota prvku menšia ako táto hodnota, prvok zlyhá pri overení obmedzenia. Ak je zadaná hodnota, min ktorá nie je platným číslom, vstup nemá žiadnu minimálnu hodnotu.

### Max
#### Maximálna hodnota

Najvyššia hodnota v rozsahu povolených hodnôt. Ak hodnota value zadaná do prvku túto hodnotu prekročí, prvok zlyhá pri overení obmedzenia. Ak hodnota atribútu max nie je číslo, prvok nemá maximálnu hodnotu.

Táto hodnota musí byť väčšia alebo rovná hodnote atribútu min. Pozrite si atribút HTML max.

### Step
Atribút step je číslo, ktoré určuje úroveň podrobnosti, ktorú musí hodnota dodržiavať. Platné sú iba hodnoty, ktoré zodpovedajú zadanému intervalu krokovania ( min ak je zadaný, inak, alebo vhodná predvolená hodnota, ak nie je zadaná žiadna z nich).

### Color

`primary`, `danger`, `complete`, `warning`, `success`

Color určuje farbu posuvníka. Ak sa hodnota nezadá, posuvnik bude mať farbu purple.

```html title="Zápis v HTML"
<wj-slider color="primary"></wj-slider>
```

Ak existuje atribut s názvom bubble, tak komponenta vykreslí nad tiahlom bublinu s vybranou hodnotou.

```html title="Zápis v HTML"
<wj-slider color="complete" min="20" max="80" value="32" step="2" bubble></wj-slider>
```

```js title="Zápis v Javascripte"
let slider = document.querySelector('wj-slider');

slider.setAttribute('color', 'complete');
slider.setAttribute('min', '20');
slider.setAttribute('max', '80');
slider.setAttribute('value', '32');
slider.setAttribute('step', '2');
slider.setAttribute('bubble', '');
```
