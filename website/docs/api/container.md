## Príklady použitia

## Atribúty a Vlastnosti

### indent

|  |  |
| --- | --- |
| Popis | Ak true, Container dostane na obrazovkách väčších ako 768px odsadenie zľava.  |
| Atribút | `indent` |
| Typ | `boolean` |
| Predvolená hodnota | `false` |

### vertical

|  |  |
| --- | --- |
| Popis | Ak true, Container dostane `flex-direcation: "column"` |
| Atribút | `vertical` |
| Typ | `boolean` |
| Predvolená hodnota | `false` |

| --- | --- |
